A strictly typed ORM library for Ethereum blockchain. It allows developers to use Ethereum blockchain as a persistent storage in an organized and model-oriented way without writing custom complex Smart contracts. The library is available for installation [via npm](https://www.npmjs.com/package/ethairballoons)
